# Tumbling Oysters

This repository contains the source for the **Tumbling Oysters** Quarto website and its published site output.

## Repository contents

- `index.qmd` and `about.qmd` define the main website pages.
- `posts/` contains the dated Quarto posts that are listed on the home page.
- `notes.qmd` and `notes.txt` store notebook-style notes content.
- `docs/` contains the rendered site that is published with GitHub Pages.
- `_quarto.yml`, `styles.css`, and supporting scripts/configuration files define the site build and presentation.
